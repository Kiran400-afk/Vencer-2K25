Web Wizards

Create innovative and responsive web applications using modern technologies

Event Description

Web Wizards is an exciting web development competition where participants showcase their skills in creating modern, responsive, and innovative web applications. This is your opportunity to demonstrate your mastery of front-end and back-end technologies while solving real-world problems through web solutions.

Rules & Guidelines

  • Teams of 2-3 members allowed
  • Duration: 24 hours
  • Participants must create a fully functional web application
  • Use of any web development framework is allowed
  • Project must be responsive and cross-browser compatible
  • Code must be original and plagiarism will lead to disqualification
  • Teams must present their project to the judges
  • Projects will be judged on innovation, design, and functionality
