Blockchain Development Challenge

Event Description

Dive into the world of blockchain technology and develop innovative decentralized applications. This competition challenges participants to create secure and efficient blockchain solutions.

Rules

  • Teams of 2-3 members are allowed
  • Duration: 10 hours
  • Participants must bring their own laptops
  • Use of Ethereum/Solidity development tools allowed
  • Smart contracts must be original
  • Test networks should be used for deployment

Requirements

  • Knowledge of blockchain fundamentals
  • Experience with Solidity programming
  • Understanding of Web3 technologies
  • Basic cryptography knowledge

Judging Criteria

  • Smart contract security (35%)
  • Innovation and usefulness (25%)
  • Code quality (20%)
  • User interface design (20%)

Prize Pool

  • 1st Place: ₹25,000
  • 2nd Place: ₹15,000
  • 3rd Place: ₹10,000
