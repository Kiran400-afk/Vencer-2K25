Data Science Challenge

Event Description

Join us for an exciting Data Science Challenge where participants will showcase their skills in data analysis, visualization, and machine learning. This competition will test your ability to derive meaningful insights from complex datasets.

Rules

  • Teams of 2-3 members are allowed
  • Duration: 6 hours
  • Participants must bring their own laptops
  • Internet access will be provided
  • Use of pre-trained models is allowed
  • Plagiarism will result in immediate disqualification

Requirements

  • Basic knowledge of Python programming
  • Familiarity with data analysis libraries (Pandas, NumPy)
  • Understanding of machine learning concepts
  • Experience with data visualization tools

Judging Criteria

  • Accuracy of analysis (40%)
  • Innovation in approach (25%)
  • Code quality and documentation (20%)
  • Visualization effectiveness (15%)

Prize Pool

  • 1st Place: ₹15,000
  • 2nd Place: ₹10,000
  • 3rd Place: ₹5,000
